# 네이버영화 상관관계 웹사이트(buggio.shop)

# 개요
네이버 영화랭킹 1 ~ 100위 영화들의 상관관계(ex. A영화를 본사람이 B영화를 본 경우, B영화를 본사람이 C,D영화를 본경우)를 시각화하여 
확인할수 있도록 만든 웹사이트

# Flask Server
 app.py 파일을 실행시키면 서버가 실행됩니다.
